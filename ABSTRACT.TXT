 
 
 DDECPP: Implement DDE Using C++ Classes
 
 DDECPP contains three sample applications--DDECli,
 DDEServ, and VBClient--which accompany the "Implementing
 DDE Using C++ Classes"msdn_ddeplus technical article in
 the Microsoft(R) Development Library.
 
 DDECli is a dynamic data exchange (DDE) client application
 implemented using a set of C++ classes. It was built with
 Microsoft Visual C++(TM) version 2.0, and tested under
 Microsoft Windows(R) NT(TM) version 3.5.
 
 DDEServ is a full DDE server application implemented using
 a set of C++ classes. It was built with Visual C++ version
 2.0, and tested under Windows NT version 3.5.
 
 VBClient is a DDE client application implemented using
 Microsoft Visual Basic(R). It was built to test the
 DDEServ server, which was implemented using C++ classes.
